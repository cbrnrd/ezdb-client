# ezdb-client

This small shard is an interface library for [ezdb](https://github.com/cbrnrd/ezdb).

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  ezdb-client:
    github: cbrnrd/ezdb-client
```

## Usage

```crystal
require "ezdb-client"
```
